# Development Setup

## Requirements

- Python 3.11+
- uv (recommended) or pip
- Git

## Setup

```bash
git clone https://github.com/saiharsha-k/rag-axis.git
cd rag-axis
uv venv
uv pip install -e ".[dev]"
```

## Running Tests

```bash
pytest tests/
```

## Running Type Checks

```bash
mypy rag_axis/
```

## Running Linting

```bash
ruff check rag_axis/
```
