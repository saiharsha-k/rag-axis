# Quickstart

<Warning>
  rag-axis is pre-alpha. Core types are in active development. This page will be updated as packages ship.
</Warning>

## Installation

```bash
pip install rag-axis
```

Requires Python 3.11+.

## What Ships in v0.0.1

The `v0.0.1` release establishes the package structure and build pipeline.
`rag_axis.core` types are in active development and will ship in `v0.1.0`.
